<span class="http">HTTP GET/POST</span>  
请求地址: https://api.datas.gq/ACGPic

# 随机二次元图
随机输出一张二次元图。

## 返回数据
```
Content-Type:image/jpeg;
```

## 参数
| 参数 | 必填 | 类型 | 说明 | 可填内容 |
| --- | --- | --- | --- | --- |
| from | 否 | string | 从何处引入。分别可以填ixiaowai（小歪API）、dmoe（樱花API），不填则从SkiMino的储存库中获取 | ixiaowai , dmoe |
