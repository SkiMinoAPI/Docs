<span class="http">HTTP GET/POST</span>  
请求地址: https://api.datas.gq/RealUrl

# 短链接还原
还原短链接指向的真实Url。

## 返回数据
```json
{"shorturl":"(ShoutUrl)","realurl":"(RealUrl)"}
```

## 参数
| 参数 | 必填 | 类型 | 说明 | 可填内容 |
| --- | --- | --- | --- | --- |
| url | 是 | string | 短链接 | * |
